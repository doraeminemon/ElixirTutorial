# Playground project for Elixir

## For install from scratch
`./bin/docker.sh`
This will install the following packages :
- brew
- asdf
- docker
## For running locally
### With docker
- Run docker container with `$ ./bin/docker.sh`
### Without docker ( local )
- Run script directly at `$ ./bin/local.sh` or
- Install with `$ asdf install`
